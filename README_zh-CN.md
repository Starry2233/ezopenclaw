### 快速上手:
对于Windows:
1. 下载[最新发行版](https://github.com/Starry2233/ezopenclaw/releases/)并解压
2. 以管理员身份运行`main.exe`

### 构建 & 运行:
1. 克隆仓库:
```bash
git clone https://github.com/Starry2233/ezopenclaw.git
```
2. 对于构建，可运行`make`或者使用`setup.py`
3. 对于直接运行项目:
```bash
python src/main.py
```
