### Usage:
For Windows:
1. Download the [latest release](https://github.com/Starry2233/ezopenclaw/releases/) and unpack it
2. Run as Administrator `main.exe`

### Build or Run:
1. Clone this repo:
```bash
git clone https://github.com/Starry2233/ezopenclaw.git
```
2. For build you can run `make` or use `setup.py`
3. For direct run:
```bash
python src/main.py
```
